門田的第四堂課 2019.11.24

  1. 練拍子：目標是「不用pick數空的點也要彈得出來」

    • 抓反拍
      • 練習：開慢的速度開始對反拍，包含4分的、8分的、16分的
      • 聽歌的時候有意識的練習抓反拍
    • 讓節奏感更進一步的方法：將16分當作兩個8分來數。培養8分音符的節奏感而非4分
    • 可以做的練習
      • 樂器
        • 第一步：唸出來節奏，會念才會彈
          *16分每一個點的唸法：
          噠 x 嗯 x = 1＊＊＊
          嗯噠嗯 x = ＊2＊＊
          嗯 x 噠 x = ＊＊3＊
          嗯 x 嗯噠 = ＊＊＊4
        • 第二步：找出相對有進步空間的節奏（例如16分的2、4）
        • 註：範圍要小，小到5-15分鐘內能夠解決
      • 練習的變化
        • 彈的音有無延音
        • 從1個音—-＞一組音（音階之類的）
        • ！！！控制的練習：用16分彈滿一組音，但要隨機的加入正在練習的點
    • Extra：使用爬格子的Pattern來練拍子

  2. 練習：

    • 再次強調使用筆記本記錄的重要性。
    • 練習的材料選擇一定要夠小，就算有很多時間也是用很多的小片段來組成整個的練習時間。
    • 重要的是規律的進步和成就感，而非快速的結果。進步需要時間。
